# 🅿️ Smart Parking AI System

Real-time parking occupancy prediction using Machine Learning.

## Features
- 🗺️ Live Birmingham City Parking Map
- 📊 Real-time Occupancy Prediction (Linear Regression)
- 🗣️ Voice Announcements
- 💳 Slot Booking System with PDF Ticket
- 🤖 AI Chatbot (English/Hindi/Urdu)
- 🔄 Auto-refresh every 10 seconds

## Tech Stack
Python | Streamlit | Scikit-learn | Plotly | Folium | gTTS | ReportLab

## Dataset
Birmingham Parking Dataset — Kaggle (35,717 records | 8 zones)

## Run Locally
pip install -r requirements.txt
streamlit run app.py
